# S-CarRepair — Interactive Vehicle Repair System for QBCore

S-CarRepair is a lightweight but feature-rich vehicle repair system designed for **QBCore-based FiveM servers**.

Players can drive into configured repair zones, interact through a clean UI prompt, and repair their vehicles using a synced progression mechanic.  
The system deducts money dynamically and plays configurable MP3 repair audio that is **audible for nearby players**, creating a more immersive world experience.

##  Features

✔ **Multiple repair zones** using PolyZone polygons  
✔ **ox_lib UI text interaction**  
✔ **QBCore progressbar integration**  
✔ Vehicle locking & freeze state while repairing  
✔ Player is **prevented from leaving the vehicle**  
✔ Automatic money handling:

✔ **Custom MP3 repair sound support**  
✔ Sound plays for **all nearby players**, not just the actor  
✔ Full localization support (+6)  
✔ Configurable sound radius per zone
